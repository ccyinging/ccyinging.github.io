Java流操作

ccying 2020-12-22

Java SE 8 stream 提供了一种在集合级别以上的概念来操作数据。

创建流

stream指串行顺序流；parallelStream指并行流。

对于并行流，只有当内存中的数据需要执行大量计算操作时才应该被使用。

  • 创建一个空流

  • 指定元素转换流

  • 将数组转换成流

  • 将集合转换成流

  • 将文件的行作为元素产生流

  • 产生一个无限流，元素值通过反复调用传入函数

  • 产生一个无限流，元素值通过传入种子数和函数，并通过反复将该函数应用到之前元素值产生

  • 产生一个流，元素按照正则表达式分割一个CharSequence对象产生

转换流

指定将初始化流转换成为其他流的中间操作，可能包含多个步骤。

流的转换操作不会修改其数据源，将是产生一个新流，它的元素派生自前一个流。

  • 产生一个流，其中包含当前流中满足条件的所有元素

  • 产生一个流，其中包含将当前流中所有元素应用于传入函数产生的结果

  • 产生一个流，其中包含当前流中最初n个元素

  • 产生一个流，其中包含当前流中除最初n个元素外的元素

  • 产生一个流，其中包含当前流中剔除重复元素后的元素

  • 产生一个流，其中包含当前流按照元素排序规则排序后的元素

  • 产生一个流，其中包含由两个流的元素连接组成的元素

  • 产生一个流，其中包含与当前流的元素相同的元素，但在每次获取一个元素时会调用传入方法

终结流

终止操作，产生结果。

  • 获取流中元素的数量

  • 获取流中的最大值

  • 获取流中的最小值

  • 获取流中第一个元素

  • 随机获取流中任意一个元素

  • 判断流中是否存在元素匹配

  • 判断流中是否存在所有元素匹配

  • 判断流中是否所有元素都不匹配

  • 获取当前流的迭代器

  • 在流的每个元素上传入函数

收集流

将流元素收集到其他数据结构中。

  • 收集到数组

  • 收集到集合

  • 连接流中所有元素，收集到字符串

  • 收集到映射表

  • 收集成SummaryStatistics对象

基本类型数据流

流库中设置有专门的基本类型数据流来直接存储基本类型值，而无需使用包装器。

short、char、byte、boolean ==> IntStream
float、double ==> DoubleStream
long ==> LongStream

适用IntStream的方法大多也适用于DoubleStream和LongStream。

基本类型数据流也是流，适用对象流的方法大多也适用基本数据类型流。
